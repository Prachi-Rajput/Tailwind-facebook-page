### `Project Clonning and Running`

Clone the project

```bash
  git clone https://github.com/Prachi-Rajput/Tailwind-facebook-page.git
```

Go to the project directory

```bash
  cd Tailwind-facebook-page
```
Install dependencies for your project

```bash
  npm install
```

Run Your Project

```bash
  npm run start
```
